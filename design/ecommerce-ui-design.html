Style guide

Modern-minimal, light. Indigo→violet accent on neutral surfaces. Font: Outfit (display) + Inter (text).

Palette

Indigo 800#312E81 · brand

Typography

Aa Display — Outfit 800
Section heading — Outfit 700

Body copy uses Inter at 14–16px with 1.55 line-height for comfortable reading across long catalog and admin tables. Muted secondary text sits at #6B7280.

Components

Category Pending Processing Shipped Delivered Cancelled

Order status colors are consistent everywhere: amber = pending, blue = processing, violet = shipped, green = delivered, red = cancelled.
